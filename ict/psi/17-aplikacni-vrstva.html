📱 Aplikační vrstva modelu ISO/OSI

Téma 17 - PSI

📡 Počítačové sítě 📚 Maturitní okruh

Aplikační vrstva je nejvyšší (7.) vrstvou v modelu ISO/OSI. Zajišťuje interakci mezi uživatelskými aplikacemi a síťovými vrstvami.

Pozice vrstvy a funkce

  • 7. pozice v modelu ISO/OSI, nad všemi ostatními vrstvami
  • Rozhraní pro aplikace: Poskytuje služby pro komunikaci přes síť
  • Sběr a zpracování dat: Kódování a šifrování dat
  • Řízení komunikace: Směrování dat mezi aplikacemi

DNS systém, WWW služba, URL

DNS (Domain Name System)

  • Účel: Překlad doménových jmen na IP adresy
  • Struktura: Hierarchická (např. .com, .org, subdomény)
  • Princip: DNS dotaz → odpověď s IP adresou

WWW služba (World Wide Web)

  • Účel: Vyhledávání a zobrazování dokumentů na internetu
  • Protokol: HTTP/HTTPS pro přenos stránek
  • Propojení stránek pomocí hypertextových odkazů

URL (Uniform Resource Locator)

  • Jedinečný identifikátor zdrojů na internetu
  • Struktura: protokol://doména/cesta?parametry

Základní protokoly

HTTP (HyperText Transfer Protocol)

  • Účel: Přenos webových stránek
  • Metody: GET, POST, PUT, DELETE
  • Stavové kódy: 200 OK, 404 Not Found, 500 Server Error

FTP (File Transfer Protocol)

  • Účel: Přenos souborů mezi klientem a serverem
  • Funkce: Nahrávání, stahování, mazání souborů
  • Dvě spojení: příkazy a přenos dat

Elektronická pošta

Účel

Odesílání a přijímání zpráv a příloh mezi zařízeními.

Protokoly

  • SMTP: Odesílání e-mailů mezi servery
  • POP3: Stahování e-mailů na klienta (zprávy se mažou ze serveru)
  • IMAP: Přístup k e-mailům na serveru bez stahování

Princip komunikace

Odesílatel → SMTP → e-mailový server → server příjemce → POP3/IMAP → příjemce

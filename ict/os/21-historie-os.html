📖 Co je operační systém?

Operační systém (OS) je základní software počítače, který zprostředkovává komunikaci mezi uživatelem, aplikacemi a hardwarem. Je to „správce" počítače.

Hlavní funkce OS:

  • Správa procesů: Spouštění programů, přidělování CPU
  • Správa paměti: Přidělování RAM aplikacím
  • Správa souborů: Organizace dat na discích
  • Správa zařízení: Komunikace s hardwarem (ovladače)
  • Uživatelské rozhraní: GUI nebo příkazová řádka
  • Bezpečnost: Oprávnění, uživatelé, ochrana

Historie operačních systémů

1. generace (1945-1955) – Bez OS

  • Počítače programovány přímo v strojovém kódu
  • Programy zadávány přepínáním přepínačů nebo děrnými štítky
  • Obsluha jedním programátorem najednou
  • Příklady: ENIAC, UNIVAC

2. generace (1955-1965) – Dávkové zpracování

  • První jednoduché OS pro automatizaci úloh
  • Dávkové zpracování (batch): Programy se shromáždily a spouštěly postupně bez zásahu obsluhy
  • Tranzistory nahradily elektronky
  • Děrné štítky a pásky jako vstup

3. generace (1965-1980) – Multiprogramování

  • Multiprogramování: V paměti více programů najednou, CPU přepíná mezi nimi
  • Timesharing: Více uživatelů sdílí počítač (terminály)
  • Integrované obvody
  • Klíčové OS:
    • MULTICS: Průkopník, inspiroval Unix
    • UNIX (1969): Ken Thompson + Dennis Ritchie (Bell Labs), psaný v C, ovlivnil vše

4. generace (1980-současnost) – Osobní počítače

  • Mikroprocesory, počítače pro jednotlivce
  • MS-DOS (1981): Microsoft pro IBM PC, příkazový řádek
  • Mac OS (1984): Apple, grafické rozhraní
  • Windows 3.0/95: Grafická nadstavba → plnohodnotný OS
  • Linux (1991): Linus Torvalds, open-source, unixový
  • Současnost: Windows 11, macOS, Linux distribuce, Android, iOS

Architektura OS – Typy jádra

Jádro (kernel) je centrální část OS, která běží v privilegovaném režimu a má přímý přístup k hardwaru.

Monolitické jádro

┌─────────────────────────────────────┐
│           UŽIVATELSKÝ PROSTOR       │
│  (aplikace, programy uživatele)     │
├─────────────────────────────────────┤
│     ┌───────────────────────────┐   │
│     │    MONOLITICKÉ JÁDRO      │   │
│     │  • Správa procesů         │   │
│     │  • Správa paměti          │   │
│     │  • Souborové systémy      │   │
│     │  • Síťové služby          │   │
│     │  • Ovladače zařízení      │   │
│     └───────────────────────────┘   │
│              KERNEL SPACE           │
└─────────────────────────────────────┘
│               HARDWARE              │
  • Vše běží v jednom velkém bloku v režimu jádra
  • Výhody: Rychlé (přímá komunikace mezi komponenty)
  • Nevýhody: Chyba v jedné části může shodit celý systém, obtížná údržba
  • Příklady: Linux, tradiční UNIX

Mikrojádro (Microkernel)

┌─────────────────────────────────────┐
│           UŽIVATELSKÝ PROSTOR       │
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│ │ Ovladače│ │Souborový│ │ Síťové  │ │
│ │ zařízení│ │ systém  │ │ služby  │ │
│ └─────────┘ └─────────┘ └─────────┘ │
├─────────────────────────────────────┤
│     ┌───────────────────────────┐   │
│     │       MIKROJÁDRO          │   │
│     │  • Základní IPC           │   │
│     │  • Správa paměti (min)    │   │
│     │  • Plánování procesů      │   │
│     └───────────────────────────┘   │
│              KERNEL SPACE           │
└─────────────────────────────────────┘
  • V jádře jen naprosté minimum, zbytek v uživatelském prostoru
  • Výhody: Stabilita (pád ovladače nezávadí systém), modularita
  • Nevýhody: Pomalejší (komunikace přes zprávy)
  • Příklady: MINIX, QNX, seL4

Hybridní jádro

  • Kombinace obou přístupů – mikrojádro s některými službami v kernel space
  • Kompromis mezi výkonem a modularitou
  • Příklady: Windows NT/10/11, macOS (XNU), BeOS

Klasifikace operačních systémů

Kritérium Typy
Podle počtu uživatelů Jednouživatelský (DOS) × Víceuživatelský (Linux, Windows Server)
Podle počtu úloh Jednoúlohový (DOS) × Víceúlohový/Multitasking (Windows, Linux)
Podle rozhraní CLI (příkazová řádka) × GUI (grafické)
Podle platformy Desktop × Server × Mobilní × Embedded × Real-time
Podle licence Proprietární (Windows) × Open-source (Linux)

Moderní operační systémy

Windows

  • Nejrozšířenější desktopový OS (~70% trhu)
  • Hybridní jádro (NT kernel od Windows NT/2000)
  • Proprietární (placený), GUI orientovaný

Linux

  • Open-source, monolitické jádro
  • Distribuce: Ubuntu, Fedora, Debian, Arch, CentOS...
  • Dominantní na serverech, superpočítačích, embedded, Android

macOS

  • Apple, založen na Darwin (BSD + XNU kernel)
  • Hybridní jádro, uzavřený ekosystém

🎓 Shrnutí pro maturitu

  • OS = správce počítače (procesy, paměť, soubory, zařízení)
  • Generace: Bez OS → Dávkové → Multiprogramování → PC
  • UNIX (1969): Základ pro Linux a macOS
  • Monolitické jádro: Vše v jednom, rychlé (Linux)
  • Mikrojádro: Minimum v jádře, stabilnější (QNX)
  • Hybridní: Kompromis (Windows, macOS)
  • Multitasking: Více úloh současně

💾 Tranzistorové elektronické paměti

Téma 4 - Hardware

🖥️ Hardware 📚 Maturitní okruh

Tranzistorové paměti využívají polovodičové tranzistory pro ukládání dat.

1. Základní dělení pamětí

  • Volatilní: Data se ztratí při odpojení napájení (RAM)
  • Non-volatilní: Data zůstávají zachována (ROM, Flash)

2. RAM (Random Access Memory)

DRAM (Dynamic RAM)

  • Jeden tranzistor + kondenzátor na bit
  • Vyžaduje refresh (kondenzátor se vybíjí)
  • Použití: Hlavní operační paměť (DDR4, DDR5)

SRAM (Static RAM)

  • 6 tranzistorů na bit (flip-flop)
  • Nepotřebuje refresh, velmi rychlá
  • Použití: Cache paměť procesoru

Typy DDR

  • DDR4: 2133-3200 MHz, 1.2V
  • DDR5: 4800-8400+ MHz, 1.1V, on-die ECC

3. ROM (Read-Only Memory)

  • Mask ROM: Data zapsána při výrobě
  • PROM: Jednorázově programovatelná
  • EPROM: Mazatelná UV světlem
  • EEPROM: Elektricky mazatelná

4. Flash paměť

  • NAND Flash: Vysoká kapacita (SSD, USB)
  • NOR Flash: Rychlé čtení (firmware)

Typy NAND buněk

  • SLC: 1 bit/buňka - nejrychlejší, nejvyšší výdrž
  • MLC: 2 bity/buňka
  • TLC: 3 bity/buňka - běžné v SSD
  • QLC: 4 bity/buňka - nejvyšší kapacita

5. Paměťová hierarchie

  1. Registry CPU: ~1 ns
  2. L1/L2/L3 Cache: 1-20 ns
  3. RAM: ~50-100 ns
  4. SSD: ~10-100 µs
  5. HDD: ~5-10 ms

🎓 Shrnutí

  • DRAM: Potřebuje refresh, RAM moduly
  • SRAM: Rychlá, cache paměť
  • Flash: SLC > MLC > TLC > QLC
  • Hierarchie: Registry → Cache → RAM → SSD

📖 Co je CSS?

Cascading Style Sheets je jazyk pro vizuální úpravu HTML dokumentů. Definuje barvy, fonty, rozložení, animace a další vzhled stránky.

💡 Oddělení struktury od vzhledu: HTML = CO je na stránce, CSS = JAK to vypadá.

Způsoby připojení CSS

<!-- 1. Inline styly (přímo v HTML) - NEdoporučuje se -->
<p style="color: red; font-size: 16px;">Červený text</p>

<!-- 2. Interní styly (v <head>) -->
<head>
    <style>
        p { color: blue; }
    </style>
</head>

<!-- 3. Externí stylopis (DOPORUČENO) -->
<head>
    <link rel="stylesheet" href="style.css">
</head>

Anatomie CSS pravidla

h1 {
    color: blue;
    font-size: 24px;
}
│    │       │
│    │       └── hodnota
│    └── vlastnost
└── selektor

Selektory

Základní selektory

/* Element selektor */
p { color: black; }                  /* Všechny <p> */

/* Třída (class) - začíná tečkou */
.upozorneni { color: red; }          /* <div class="upozorneni"> */

/* ID - začíná # (unikátní na stránce) */
#hlavni-nadpis { font-size: 32px; }  /* <h1 id="hlavni-nadpis"> */

/* Univerzální selektor */
* { margin: 0; padding: 0; }         /* Všechny elementy */

Kombinované selektory

/* Potomek (mezera) */
article p { color: gray; }           /* <p> uvnitř <article> */

/* Přímý potomek (>) */
ul > li { list-style: none; }        /* Přímé <li> v <ul> */

/* Sourozenec (+) */
h2 + p { font-weight: bold; }        /* <p> hned za <h2> */

/* Více selektorů (,) */
h1, h2, h3 { font-family: Arial; }   /* Všechny nadpisy */

/* Kombinace element + třída */
p.dulezity { color: red; }           /* <p class="dulezity"> */

Pseudotřídy a pseudoelementy

/* Pseudotřídy - stav elementu */
a:hover { color: red; }              /* Při najetí myší */
a:visited { color: purple; }         /* Navštívený odkaz */
a:active { color: orange; }          /* Při kliknutí */
input:focus { border-color: blue; }  /* Když je aktivní */

button:disabled { opacity: 0.5; }    /* Zakázané tlačítko */
li:first-child { font-weight: bold; }/* První položka seznamu */
li:nth-child(2n) { background: #eee; }/* Sudé položky */

/* Pseudoelementy - část elementu */
p::first-line { font-variant: small-caps; }
p::first-letter { font-size: 2em; }
h2::before { content: "→ "; }        /* Přidá obsah před */
h2::after { content: " ←"; }         /* Přidá obsah za */

Box Model

Každý HTML element je „krabice" se 4 vrstvami:

┌─────────────── margin ─────────────────┐
│ ┌─────────── border ───────────────┐   │
│ │ ┌─────── padding ─────────────┐  │   │
│ │ │ ┌────── content ──────────┐ │  │   │
│ │ │ │                         │ │  │   │
│ │ │ │    Obsah elementu       │ │  │   │
│ │ │ │                         │ │  │   │
│ │ │ └─────────────────────────┘ │  │   │
│ │ └─────────────────────────────┘  │   │
│ └──────────────────────────────────┘   │
└────────────────────────────────────────┘
div {
    /* Obsah */
    width: 200px;
    height: 100px;
    
    /* Vnitřní odsazení */
    padding: 20px;                    /* Všechny strany */
    padding: 10px 20px;               /* Vertikální | Horizontální */
    padding: 10px 20px 15px 5px;      /* Top Right Bottom Left */
    
    /* Ohraničení */
    border: 2px solid black;          /* Šířka Styl Barva */
    border-radius: 8px;               /* Zaoblené rohy */
    
    /* Vnější odsazení */
    margin: 20px;
    margin: 0 auto;                   /* Vycentruje horizontálně */
    
    /* DŮLEŽITÉ: box-sizing */
    box-sizing: border-box;           /* width zahrnuje padding a border */
}

💡 box-sizing: border-box; je téměř vždy lepší volba – šířka elementu zůstane stejná i s paddingem a borderem.

Barvy a pozadí

div {
    /* Barvy */
    color: red;                       /* Pojmenovaná */
    color: #ff0000;                   /* HEX */
    color: rgb(255, 0, 0);            /* RGB */
    color: rgba(255, 0, 0, 0.5);      /* RGBA s průhledností */
    color: hsl(0, 100%, 50%);         /* HSL */
    
    /* Pozadí */
    background-color: #f0f0f0;
    background-image: url('obrazek.jpg');
    background-size: cover;           /* Vyplní celý element */
    background-position: center;
    background-repeat: no-repeat;
    
    /* Zkrácený zápis */
    background: #f0f0f0 url('bg.jpg') no-repeat center/cover;
    
    /* Gradient */
    background: linear-gradient(to right, #ff0000, #0000ff);
    background: radial-gradient(circle, #fff, #000);
}

Typography

p {
    font-family: 'Arial', sans-serif; /* Záložní fonty */
    font-size: 16px;                  /* Velikost */
    font-weight: bold;                /* Tloušťka (400 = normal, 700 = bold) */
    font-style: italic;               /* Styl */
    
    line-height: 1.5;                 /* Výška řádku */
    letter-spacing: 1px;              /* Mezery mezi písmeny */
    text-align: center;               /* left | center | right | justify */
    text-decoration: underline;       /* none | underline | line-through */
    text-transform: uppercase;        /* uppercase | lowercase | capitalize */
}

Flexbox (1D layout)

Moderní způsob zarovnání elementů v jednom směru (řádku nebo sloupci).

.kontejner {
    display: flex;                    /* Aktivuje flexbox */
    flex-direction: row;              /* row | column | row-reverse */
    
    /* Zarovnání na hlavní ose */
    justify-content: center;          /* flex-start | center | flex-end */
                                      /* space-between | space-around | space-evenly */
    
    /* Zarovnání na vedlejší ose */
    align-items: center;              /* flex-start | center | flex-end | stretch */
    
    flex-wrap: wrap;                  /* Zalamování na nový řádek */
    gap: 20px;                        /* Mezery mezi potomky */
}

.polozka {
    flex: 1;                          /* Roztáhne se, zabere 1 díl */
    flex-grow: 1;                     /* Růst */
    flex-shrink: 0;                   /* Nesmršťuje se */
    flex-basis: 200px;                /* Výchozí velikost */
}

CSS Grid (2D layout)

Pro komplexní dvourozměrné rozvržení (řádky i sloupce).

.grid-kontejner {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;  /* 3 sloupce (fr = podíl) */
    grid-template-rows: 100px auto;       /* 2 řádky */
    gap: 20px;                            /* Mezery */
}

.polozka {
    grid-column: 1 / 3;                   /* Od sloupce 1 do 3 */
    grid-row: 1 / 2;                      /* Pouze první řádek */
}

Responzivní design (Media Queries)

/* Základní styl (mobile-first) */
.kontejner {
    padding: 10px;
}

/* Tablety (768px a více) */
@media (min-width: 768px) {
    .kontejner {
        padding: 20px;
    }
}

/* Desktop (1024px a více) */
@media (min-width: 1024px) {
    .kontejner {
        max-width: 1200px;
        margin: 0 auto;
    }
}

Specificita (priorita pravidel)

Když se pravidla střetnou, vyhrává specifičtější selektor:

  1. !important (vyhýbej se!)
  2. Inline styly (style="")
  3. ID selektory (#id)
  4. Třídy, pseudotřídy (.class, :hover)
  5. Elementy (p, div)

🎓 Shrnutí pro maturitu

  • CSS = stylovací jazyk pro vzhled HTML
  • Selektory: element, .třída, #id, kombinace
  • Box model: content → padding → border → margin
  • box-sizing: border-box = width zahrnuje vše
  • Flexbox: 1D layout (display: flex, justify-content, align-items)
  • Grid: 2D layout (display: grid, grid-template-columns)
  • Media queries: @media (min-width: 768px) pro responzivitu
  • Pseudotřídy: :hover, :focus, :nth-child()

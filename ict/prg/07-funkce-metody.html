⚙️ Funkce a metody

Téma 7 - Programování

💻 Programování 📚 Maturitní okruh

📖 Co je funkce a proč ji používáme?

Funkce je pojmenovaný blok kódu, který provádí určitou úlohu. Může přijímat vstupní hodnoty (parametry) a vracet výsledek.

💡 Proč používat funkce?

  • Znovupoužitelnost: Napíšeš jednou, voláš opakovaně
  • Přehlednost: Dlouhý kód rozdělíš na menší, pojmenované části
  • Údržba: Opravu uděláš na jednom místě
  • Testovatelnost: Snazší testování izolovaných částí

Terminologie:

  • Funkce: Obecný pojem (v C, C++, samostatná)
  • Metoda: Funkce, která patří do třídy/objektu (OOP)
  • Procedura: Funkce, která nic nevrací (void)

Definice a volání funkce

// Syntaxe definice:
// návratový_typ název_funkce(parametry) {
//     tělo funkce
//     return hodnota;  // pokud není void
// }

// Funkce s návratovou hodnotou
int secti(int a, int b) {
    return a + b;
}

// Funkce bez návratové hodnoty (procedura)
void pozdrav() {
    cout << "Ahoj!" << endl;
    // není return (nebo prázdný return;)
}

// Volání funkcí
int vysledek = secti(5, 3);  // vysledek = 8
pozdrav();                    // Vypíše: Ahoj!

Průběh volání funkce

  1. Program narazí na volání funkce secti(5, 3)
  2. Argumenty 5 a 3 se předají do parametrů a a b
  3. Vykoná se tělo funkce
  4. return vrátí hodnotu 8 na místo volání
  5. Program pokračuje za voláním

Předávání parametrů

Způsob, jakým se hodnoty „přenesou" do funkce. Klíčový koncept s různým chováním.

Předání hodnotou (by value)

Funkce dostane kopii hodnoty. Originál zůstane nezměněn.

void zdvojnasob(int x) {
    x = x * 2;      // Mění jen lokální kopii!
    cout << "Uvnitř: " << x << endl;  // 10
}

int cislo = 5;
zdvojnasob(cislo);
cout << "Venku: " << cislo << endl;   // 5 - nezměněno!

Předání odkazem (by reference)

Funkce pracuje přímo s originální proměnnou. Změny se projeví i vně.

// C++ - reference (&)
void zdvojnasob(int& x) {    // & = reference
    x = x * 2;               // Mění originál!
}

int cislo = 5;
zdvojnasob(cislo);
cout << cislo;               // 10 - změněno!

// C# - ref keyword
void Zdvojnasob(ref int x) {
    x = x * 2;
}

int cislo = 5;
Zdvojnasob(ref cislo);       // Musíš napsat ref i při volání
Console.WriteLine(cislo);    // 10

Výstupní parametry (C# out)

Pro funkce, které potřebují vrátit více hodnot:

void Rozdel(int cislo, out int podil, out int zbytek) {
    podil = cislo / 10;
    zbytek = cislo % 10;
}

// Použití:
int p, z;
Rozdel(42, out p, out z);
Console.WriteLine($"Podíl: {p}, Zbytek: {z}");  // Podíl: 4, Zbytek: 2

// Rozdíl ref vs out:
// ref - musí být inicializovaná PŘED voláním
// out - musí být přiřazena UVNITŘ funkce

Návratové hodnoty

// void - funkce nic nevrací
void Vypis(string text) {
    Console.WriteLine(text);
    // return;  // volitelný prázdný return
}

// Návratová hodnota - musí odpovídat typu
int Mocnina(int zaklad, int exponent) {
    int vysledek = 1;
    for (int i = 0; i < exponent; i++) {
        vysledek *= zaklad;
    }
    return vysledek;  // MUSÍ být return s hodnotou
}

// return také ukončuje funkci
int Prvni Kladny(int[] cisla) {
    foreach (int c in cisla) {
        if (c > 0) return c;  // Okamžitě ukončí a vrátí
    }
    return -1;  // Kdyby žádné kladné nebylo
}

Přetěžování funkcí (Function Overloading)

Více funkcí se stejným názvem, ale různými parametry. Kompilátor vybere správnou verzi podle argumentů.

// Stejný název, různé typy parametrů
int Secti(int a, int b) {
    return a + b;
}

double Secti(double a, double b) {
    return a + b;
}

// Různý počet parametrů
int Secti(int a, int b, int c) {
    return a + b + c;
}

// Volání - kompilátor vybere správnou verzi
Secti(5, 3);        // Volá int verzi → 8
Secti(2.5, 1.5);    // Volá double verzi → 4.0
Secti(1, 2, 3);     // Volá tříparametrovou → 6

⚠️ Pozor: Nelze přetížit jen změnou návratového typu!

Výchozí hodnoty parametrů

Parametr může mít výchozí hodnotu – pak ho nemusíš při volání uvádět.

void Pozdrav(string jmeno = "uživateli", int kolik = 1) {
    for (int i = 0; i < kolik; i++) {
        cout << "Ahoj, " << jmeno << "!" << endl;
    }
}

// Různé způsoby volání:
Pozdrav();                    // "Ahoj, uživateli!" (1x)
Pozdrav("Jane");              // "Ahoj, Jane!" (1x)
Pozdrav("Jane", 3);           // "Ahoj, Jane!" (3x)

⚠️ Pravidlo: Parametry s výchozími hodnotami musí být na konci seznamu!

Rekurze

Funkce, která volá samu sebe. Musí mít ukončovací podmínku, jinak vznikne nekonečná smyčka.

// Výpočet faktoriálu: n! = n × (n-1) × ... × 1
int Faktorial(int n) {
    // Ukončovací podmínka (base case)
    if (n <= 1) return 1;
    
    // Rekurzivní volání
    return n * Faktorial(n - 1);
}

// Jak to funguje:
// Faktorial(5)
// = 5 * Faktorial(4)
// = 5 * 4 * Faktorial(3)
// = 5 * 4 * 3 * Faktorial(2)
// = 5 * 4 * 3 * 2 * Faktorial(1)
// = 5 * 4 * 3 * 2 * 1
// = 120

// Fibonacci - každé číslo je součtem dvou předchozích
int Fibonacci(int n) {
    if (n <= 1) return n;
    return Fibonacci(n - 1) + Fibonacci(n - 2);
}

⚠️ Pozor na rekurzi:

  • Bez ukončovací podmínky → Stack Overflow
  • Může být pomalejší než iterace
  • Pro velká čísla může vyčerpat paměť zásobníku

Lokální vs globální proměnné

int globalni = 100;       // Globální - viditelná všude

void MojeFunkce() {
    int lokalni = 50;     // Lokální - jen uvnitř funkce
    
    cout << globalni;     // OK - vidí globální
    cout << lokalni;      // OK - vidí lokální
}

void JinaFunkce() {
    cout << globalni;     // OK
    cout << lokalni;      // CHYBA! Nevidí lokální z jiné funkce
}

💡 Dobré praktiky: Používej lokální proměnné, globálním se vyhýbej. Předávej data jako parametry.

🎓 Shrnutí pro maturitu

  • Funkce = pojmenovaný blok kódu, lze opakovaně volat
  • Syntaxe: typ název(parametry) { tělo; return; }
  • Předání hodnotou: Kopie – originál nezměněn
  • Předání odkazem: Reference (&, ref) – mění originál
  • void: Funkce bez návratové hodnoty
  • Přetěžování: Stejné jméno + různé parametry
  • Rekurze: Funkce volá samu sebe (MUSÍ mít ukončení)

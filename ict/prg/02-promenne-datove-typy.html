📊 Proměnné a datové typy

Téma 2 - Programování

💻 Programování 📚 Maturitní okruh

📖 Co je proměnná?

Proměnná je pojmenované místo v paměti, kde program ukládá data. Představ si ji jako krabici s nálepkou – krabice má jméno (název proměnné), typ (jakou velikost má krabice) a obsah (hodnota).

Každá proměnná má tři základní vlastnosti:

  • Název (identifikátor): Jak proměnnou v kódu oslovujeme
  • Datový typ: Určuje, jaká data může proměnná obsahovat a kolik paměti zabírá
  • Hodnota: Aktuální obsah proměnné

Deklarace a inicializace

Než můžeme proměnnou použít, musíme ji deklarovat (oznámit kompilátoru, že existuje) a typicky také inicializovat (přiřadit jí počáteční hodnotu).

// Deklarace - vytvoření proměnné bez hodnoty
int cislo;

// Inicializace - přiřazení hodnoty
cislo = 5;

// Deklarace s inicializací (nejčastější způsob)
int vek = 25;
string jmeno = "Jan";
double pi = 3.14159;

⚠️ Pozor: V C++ neinicializovaná proměnná obsahuje „odpadky" – náhodná data, která tam zůstala v paměti. V C# kompilátor použití neinicializované proměnné nedovolí.

Pravidla pojmenování proměnných

  • Název musí začínat písmenem nebo podtržítkem (_)
  • Může obsahovat písmena, číslice a podtržítka
  • Nesmí být klíčové slovo jazyka (int, if, while, class...)
  • Rozlišují se velká a malá písmena (cislo ≠ Cislo ≠ CISLO)

💡 Konvence pojmenování:

  • camelCase: první písmeno malé, další slova velká (pocetStudentu) – C#, Java pro proměnné
  • PascalCase: všechna slova začínají velkým (PocetStudentu) – C# pro třídy a metody
  • snake_case: slova oddělená podtržítkem (pocet_studentu) – Python, C

Primitivní datové typy

Primitivní typy jsou základní stavební bloky – ukládají jednoduchou hodnotu přímo v paměti.

Celočíselné typy

Pro ukládání celých čísel (bez desetinné čárky). Liší se velikostí a rozsahem hodnot:

Typ Velikost Rozsah Použití
byte 1 bajt 0 až 255 Malá kladná čísla, bajty dat
short 2 bajty -32 768 až 32 767 Menší čísla, úspora paměti
int 4 bajty ±2,1 miliardy Nejčastější – věk, počty
long 8 bajtů ±9 trilionů Velká čísla, ID, časová razítka

Desetinné typy (s plovoucí řádovou čárkou)

Pro čísla s desetinnou částí. Pozor na nepřesnost při výpočtech!

  • float: 4 bajty, přesnost ~7 číslic – používá se zřídka
  • double: 8 bajtů, přesnost ~15 číslic – nejčastější pro desetinná čísla
  • decimal (C#): 16 bajtů, vysoká přesnost – pro finance a peníze 
float teplota = 36.6f;      // 'f' suffix pro float
double pi = 3.141592653589;
decimal cena = 199.99m;     // 'm' suffix pro decimal v C#

⚠️ Proč decimal pro peníze? Protože float a double mají zaokrouhlovací chyby:

double a = 0.1 + 0.2;  
Console.WriteLine(a);  // Vypíše 0.30000000000000004 !

Další základní typy

  • char: Jeden znak v apostrofech – 'A', '5', 'ř'
  • bool: Logická hodnota – pouze true nebo false
  • string: Řetězec znaků v uvozovkách – "Ahoj světe"
char pismeno = 'A';
char cisloJakoZnak = '5';   // pozor: '5' ≠ 5

bool jeStudent = true;
bool maRidicskyPrukaz = false;

string pozdrav = "Ahoj!";
string prazdny = "";        // prázdný řetězec
string s = null;            // žádný řetězec (reference)

Typová konverze

Někdy potřebujeme převést hodnotu z jednoho typu na jiný. Existují dva způsoby:

Implicitní konverze (automatická)

Probíhá automaticky, když nehrozí ztráta dat – z menšího typu do většího:

int cislo = 100;
long velkeCislo = cislo;    // OK: int → long automaticky
double des = cislo;         // OK: int → double automaticky

Explicitní konverze (casting)

Musíme ji napsat ručně, protože může dojít ke ztrátě dat:

double pi = 3.14159;
int cele = (int)pi;         // cele = 3 (ztráta desetinné části!)

long velke = 5000000000;
int male = (int)velke;      // POZOR: přetečení, špatný výsledek!

Konverze řetězců

// Číslo → String
int cislo = 42;
string text = cislo.ToString();  // "42"

// String → Číslo (C#)
string vstup = "123";
int hodnota = int.Parse(vstup);           // Vyhodí výjimku při chybě
int hodnota2 = Convert.ToInt32(vstup);    // Alternativa

// Bezpečná konverze s TryParse
if (int.TryParse(vstup, out int vysledek)) {
    Console.WriteLine("Číslo: " + vysledek);
} else {
    Console.WriteLine("Neplatný vstup!");
}

Konstanty

Hodnota, která se nemění po celou dobu běhu programu:

const double PI = 3.14159;
const int MAX_POKUSU = 3;

PI = 3.14;  // CHYBA! Konstanta nelze změnit

💡 Kdy použít konstantu? Pro hodnoty, které mají význam a nemění se: matematické konstanty, limity, konfigurace.

Příklady v C++ vs C#

// C++
#include <iostream>
#include <string>
using namespace std;

int vek = 25;
double vyska = 180.5;
char znamka = 'A';
bool absolvent = false;
string jmeno = "Petr";

cout << jmeno << " má " << vek << " let" << endl;
// C#
int vek = 25;
double vyska = 180.5;
char znamka = 'A';
bool absolvent = false;
string jmeno = "Petr";

Console.WriteLine($"{jmeno} má {vek} let");

🎓 Shrnutí pro maturitu

  • Proměnná = pojmenované místo v paměti (název + typ + hodnota)
  • Deklarace = vytvoření, inicializace = přiřazení hodnoty
  • Pro celá čísla nejčastěji int, pro desetinná double
  • Pro peníze vždy decimal! (kvůli přesnosti)
  • Implicitní konverze = automatická, explicitní = ruční s rizikem ztráty dat
  • const = neměnná hodnota

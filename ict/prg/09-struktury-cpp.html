📖 Co je struktura a proč ji používáme?

Struktura (struct) je uživatelsky definovaný datový typ, který umožňuje seskupit související data různých typů do jednoho celku.

💡 Představ si to takto: Chceš evidovat informace o osobě – jméno, věk a výšku. Bez struktury bys potřeboval tři samostatné proměnné. Se strukturou vytvoříš jeden „balíček", který tyto údaje spojuje dohromady.

Proč používat struktury?

  • Organizace dat: Související data jsou pohromadě
  • Čitelnost kódu: Je jasné, že student.jmeno patří ke studentovi
  • Snadnější předávání: Místo 5 parametrů předáš jednu strukturu
  • Základ pro OOP: Struktury jsou předchůdcem tříd

Praktické příklady použití

  • Evidence záznamů: Student, Zaměstnanec, Produkt
  • Geometrie: Bod (x, y), Obdélník (x, y, šířka, výška)
  • Herní vývoj: Pozice hráče, statistiky postavy
  • Data z formulářů: Registrační údaje, objednávka

Definice struktury

Strukturu definujeme pomocí klíčového slova struct, následovaného názvem a členskými proměnnými ve složených závorkách:

// Definice struktury - popisuje "šablonu"
struct Osoba {
    string jmeno;    // členská proměnná pro jméno
    int vek;         // členská proměnná pro věk
    double vyska;    // členská proměnná pro výšku v cm
};  // ⚠️ Nezapomeň na středník za závorkou!

⚠️ Častá chyba: Zapomenutí středníku za uzavírací závorkou }; způsobí matoucí chybové hlášky kompilátoru!

Vytvoření instance (proměnné typu struktura)

Definice struktury je jen „šablona". Pro uložení dat musíme vytvořit instanci:

// Vytvoření instance a postupné přiřazení hodnot
Osoba jan;              // Vytvoříme proměnnou typu Osoba
jan.jmeno = "Jan Novák";
jan.vek = 25;
jan.vyska = 180.5;

// Inicializace při vytvoření (kratší způsob)
Osoba eva = {"Eva Svobodová", 30, 165.0};

// C++11: Pojmenovaná inicializace (přehlednější)
Osoba petr = {
    .jmeno = "Petr Černý",
    .vek = 28,
    .vyska = 175.0
};

Přístup k členům

K jednotlivým položkám struktury přistupujeme pomocí tečkové notace:

Osoba jan = {"Jan Novák", 25, 180.5};

// Čtení hodnot
cout << "Jméno: " << jan.jmeno << endl;      // Jan Novák
cout << "Věk: " << jan.vek << " let" << endl; // 25 let

// Změna hodnot
jan.vek++;       // Jan měl narozeniny
jan.vyska = 181.0;

// Výpočty s členy
if (jan.vek >= 18) {
    cout << jan.jmeno << " je plnoletý." << endl;
}

Pole struktur

Struktury se často používají v polích pro uložení více záznamů stejného typu:

// Pole 3 osob
Osoba lide[3];

// Naplnění dat
lide[0] = {"Jan", 25, 180.0};
lide[1] = {"Eva", 30, 165.0};
lide[2] = {"Petr", 35, 175.0};

// Procházení pole struktur
for (int i = 0; i < 3; i++) {
    cout << lide[i].jmeno << " (" << lide[i].vek << " let)" << endl;
}

// Výstup:
// Jan (25 let)
// Eva (30 let)
// Petr (35 let)

Struktury a funkce

Struktury můžeme předávat funkcím jako parametry a vracet z nich:

Předání struktury do funkce

// Předání hodnotou (vytvoří se kopie)
void vypisOsobu(Osoba o) {
    cout << o.jmeno << ", " << o.vek << " let" << endl;
}

// Předání referencí (efektivnější, nemění originál)
void vypisOsobuRef(const Osoba& o) {
    cout << o.jmeno << ", " << o.vek << " let" << endl;
}

// Předání referencí s možností změny
void oslavNarozeniny(Osoba& o) {
    o.vek++;
    cout << o.jmeno << " má nyní " << o.vek << " let!" << endl;
}

Vrácení struktury z funkce

Osoba vytvorOsobu(string jmeno, int vek, double vyska) {
    Osoba nova;
    nova.jmeno = jmeno;
    nova.vek = vek;
    nova.vyska = vyska;
    return nova;
}

// Použití
Osoba student = vytvorOsobu("Milan", 20, 178.0);
vypisOsobu(student);  // Milan, 20 let

Vnořené struktury

Struktura může obsahovat jinou strukturu jako svého člena:

struct Adresa {
    string ulice;
    string mesto;
    int psc;
};

struct Zamestnanec {
    string jmeno;
    double plat;
    Adresa bydliste;   // vnořená struktura
};

// Použití
Zamestnanec karel;
karel.jmeno = "Karel Dvořák";
karel.plat = 45000;
karel.bydliste.ulice = "Hlavní 123";
karel.bydliste.mesto = "Praha";
karel.bydliste.psc = 11000;

cout << karel.jmeno << " bydlí v městě " << karel.bydliste.mesto;

Rozdíl mezi struct a class

V C++ jsou struct a class téměř identické. Jediný rozdíl je ve výchozí viditelnosti členů:

Vlastnost struct class
Výchozí přístup public private
Může mít metody Ano Ano
Může dědit Ano Ano

💡 Konvence:

  • struct: Pro jednoduché datové typy bez složité logiky (POD – Plain Old Data)
  • class: Pro složitější objekty s metodami, konstruktory a zapouzdřením

Struktura s metodami (C++)

V C++ může mít struktura i metody, podobně jako třída:

struct Obdelnik {
    double sirka;
    double vyska;
    
    // Metoda pro výpočet obsahu
    double obsah() {
        return sirka * vyska;
    }
    
    // Metoda pro výpočet obvodu
    double obvod() {
        return 2 * (sirka + vyska);
    }
};

// Použití
Obdelnik o = {5.0, 3.0};
cout << "Obsah: " << o.obsah() << endl;    // 15
cout << "Obvod: " << o.obvod() << endl;    // 16

  • Struktura = uživatelský datový typ seskupující různé datové typy
  • Definice: struct Nazev { členy }; – pozor na středník na konci!
  • Přístup k členům pomocí tečky: osoba.jmeno
  • Lze vytvářet pole struktur pro více záznamů
  • struct vs class: Výchozí viditelnost (public vs private)
  • Konvence: struct pro data, class pro objekty s logikou

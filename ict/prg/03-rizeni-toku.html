🔀 Řízení toku programu

Téma 3 - Programování

💻 Programování 📚 Maturitní okruh

📖 Co je řízení toku?

Normálně se příkazy vykonávají postupně – jeden za druhým. Řízení toku nám umožňuje tuto posloupnost měnit: některé příkazy přeskočit, jiné opakovat, nebo zvolit jednu z několika možností.

Základní struktury řízení toku jsou:

  • Podmíněné příkazy: Vykonají se jen když je splněna podmínka (if, switch)
  • Cykly: Opakují příkazy (for, while) – viz téma 4
  • Skoky: Přeruší normální tok (break, continue, return)

Podmínka if-else

Nejzákladnější podmínka – pokud je výraz pravdivý, vykoná se blok kódu. Pokud ne, může se vykonat alternativa.

Jednoduchý if

int vek = 18;

if (vek >= 18) {
    cout << "Jsi plnoletý, můžeš volit." << endl;
}
// Pokud podmínka není splněna, nic se nestane

if-else (dvě větve)

Když chceme udělat něco jiného, pokud podmínka neplatí:

int vek = 16;

if (vek >= 18) {
    cout << "Jsi plnoletý.";
} else {
    cout << "Jsi nezletilý.";
}
// Vždy se vykoná PŘESNĚ jeden z bloků

else if (více větví)

Pro více možností testujeme podmínky postupně – první splněná „vyhrává":

int znamka = 2;

if (znamka == 1) {
    cout << "Výborně!";
} else if (znamka == 2) {
    cout << "Chvalitebně";
} else if (znamka == 3) {
    cout << "Dobře";
} else if (znamka == 4) {
    cout << "Dostatečně";
} else if (znamka == 5) {
    cout << "Nedostatečně";
} else {
    cout << "Neplatná známka";  // záchytná větev
}

⚠️ Pozor na pořadí! Podmínky se testují shora dolů. První splněná se vykoná a zbytek se přeskočí.

Příkaz switch

Elegantní způsob, jak porovnat jednu hodnotu s více možnostmi. Vhodný pro diskrétní hodnoty (čísla, znaky, enumy).

int den = 3;  // 1 = pondělí, 2 = úterý, ...

switch (den) {
    case 1:
        cout << "Pondělí";
        break;
    case 2:
        cout << "Úterý";
        break;
    case 3:
        cout << "Středa";
        break;
    case 4:
        cout << "Čtvrtek";
        break;
    case 5:
        cout << "Pátek";
        break;
    case 6:
    case 7:
        cout << "Víkend!";  // case 6 i 7 sdílejí tento kód
        break;
    default:
        cout << "Neplatný den";
}

⚠️ Důležité: break! Bez break by kód „propadl" do dalšího case. To se nazývá fall-through a většinou to není žádoucí.

Kdy použít switch vs if-else?

  • switch: Když porovnáváme jednu proměnnou s mnoha konkrétními hodnotami
  • if-else: Když máme složitější podmínky, rozsahy, nebo porovnáváme více proměnných

Relační operátory

Porovnávají dvě hodnoty a vrací true nebo false:

Operátor Význam Příklad
== Rovná se x == 5
!= Nerovná se x != 0
< Menší než vek < 18
> Větší než teplota > 30
<= Menší nebo rovno body <= 100
>= Větší nebo rovno vek >= 18

⚠️ Nejčastější chyba: Záměna = (přiřazení) za == (porovnání)!

// ŠPATNĚ - přiřazení, vždy true!
if (x = 5) { ... }

// SPRÁVNĚ - porovnání
if (x == 5) { ... }

Logické operátory

Kombinují více podmínek do jedné:

AND (&&) – obě podmínky musí platit

int vek = 25;
bool maRidicsak = true;

// Může řídit jen kdo je plnoletý A má řidičák
if (vek >= 18 && maRidicsak) {
    cout << "Můžeš řídit auto.";
}

OR (||) – alespoň jedna podmínka musí platit

bool jeStudent = true;
bool jeSenior = false;

// Slevu má student NEBO senior
if (jeStudent || jeSenior) {
    cout << "Máš nárok na slevu.";
}

NOT (!) – negace podmínky

bool jeZamceno = true;

if (!jeZamceno) {
    cout << "Dveře jsou otevřené.";
} else {
    cout << "Dveře jsou zamčené.";
}

Kombinace operátorů

// Složitější podmínka
if ((vek >= 18 && vek <= 65) || jeInvalida) {
    cout << "Může pracovat.";
}
// Závorky určují prioritu - vždy je používejte pro jasnost!

Ternární operátor

Zkrácený zápis jednoduchého if-else, který vrací hodnotu. Čte se jako otázka:

// Syntaxe: podmínka ? hodnota_pokud_true : hodnota_pokud_false

int vek = 20;
string status = (vek >= 18) ? "dospělý" : "nezletilý";
// Výsledek: status = "dospělý"

// Je ekvivalentní:
string status;
if (vek >= 18) {
    status = "dospělý";
} else {
    status = "nezletilý";
}

💡 Kdy použít? Pouze pro jednoduché případy. U složitějších podmínek je if-else čitelnější.

Vnořené podmínky

Podmínky můžeme vnořovat do sebe, ale pozor na přílišnou složitost:

bool jeStudent = true;
double prumer = 1.3;

if (jeStudent) {
    if (prumer < 1.5) {
        cout << "Prospěl s vyznamenáním";
    } else if (prumer <= 4.0) {
        cout << "Prospěl";
    } else {
        cout << "Neprospěl";
    }
} else {
    cout << "Není student";
}

⚠️ Tip: Příliš hluboké vnoření (více než 2-3 úrovně) snižuje čitelnost. Použij návratové příkazy nebo refaktoruj do funkcí.

🎓 Shrnutí pro maturitu

  • if-else: Základní větvení, vykoná se jedna z větví
  • switch: Pro porovnání s mnoha hodnotami, nezapomeň na break!
  • Relační op.: ==, !=, <, >, <=, >=
  • Logické op.: && (AND), || (OR), ! (NOT)
  • Ternární operátor: podmínka ? true : false
  • Častá chyba: = vs ==

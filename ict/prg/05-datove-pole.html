📖 Co je pole (Array)?

Pole je datová struktura obsahující kolekci prvků stejného typu, uložených v souvislé paměti. K prvkům přistupujeme pomocí indexu.

💡 Představ si to jako řadu krabiček:

Index:    [0]   [1]   [2]   [3]   [4]
Hodnoty:   10    20    30    40    50

Klíčové vlastnosti pole:

  • Indexování od 0: První prvek má index 0, ne 1!
  • Pevná velikost: Po vytvoření nelze měnit počet prvků (v klasickém poli)
  • Rychlý přístup: K libovolnému prvku se dostaneme okamžitě pomocí indexu
  • Homogenní: Všechny prvky musí být stejného typu

Pole v C++

Deklarace a inicializace

// Deklarace bez inicializace - POZOR: obsahuje náhodná data!
int cisla[5];

// Inicializace při deklaraci
int cisla[5] = {10, 20, 30, 40, 50};

// Automatické určení velikosti podle počtu hodnot
int cisla[] = {10, 20, 30, 40, 50};  // velikost = 5

// Částečná inicializace (zbytek bude 0)
int cisla[5] = {10, 20};  // = {10, 20, 0, 0, 0}

// Inicializace na samé nuly
int cisla[5] = {0};       // = {0, 0, 0, 0, 0}
int cisla[5] = {};        // totéž v C++11

Přístup k prvkům

int cisla[5] = {10, 20, 30, 40, 50};

// Čtení hodnoty
int prvni = cisla[0];    // = 10
int posledni = cisla[4]; // = 50

// Zápis hodnoty
cisla[0] = 100;          // pole je nyní {100, 20, 30, 40, 50}

// ⚠️ Častá chyba: přístup mimo meze!
int spatne = cisla[5];   // CHYBA! Index 5 neexistuje (max je 4)
// C++ to NEZASTAVÍ - přečteš náhodnou paměť nebo program spadne

⚠️ Index out of bounds: C++ nekontroluje přístup mimo meze pole. Špatný index může způsobit pád programu, poškození dat nebo bezpečnostní díry!

Procházení pole

int cisla[5] = {10, 20, 30, 40, 50};

// Klasický for cyklus (nejčastější způsob)
for (int i = 0; i < 5; i++) {
    cout << cisla[i] << " ";
}
// Výstup: 10 20 30 40 50

// Range-based for (C++11) - jednodušší, bezpečnější
for (int cislo : cisla) {
    cout << cislo << " ";
}
// Výstup: 10 20 30 40 50

// S možností modifikace (reference)
for (int& cislo : cisla) {
    cislo *= 2;  // zdvojnásobí každý prvek
}

Zjištění velikosti pole v C++

int cisla[] = {10, 20, 30, 40, 50};

// Starý způsob (sizeof)
int velikost = sizeof(cisla) / sizeof(cisla[0]);  // = 5

// Moderní způsob (C++17)
#include 
std::array arr = {10, 20, 30, 40, 50};
int vel = arr.size();  // = 5

Pole v C#

Deklarace a inicializace

// Deklarace a vytvoření (všechny prvky = 0)
int[] cisla = new int[5];

// Inicializace s hodnotami
int[] cisla = {10, 20, 30, 40, 50};

// Explicitní forma
int[] cisla = new int[] {10, 20, 30, 40, 50};

// Prázdné pole pozdější inicializace
int[] cisla;
cisla = new int[5];

Vlastnost Length

int[] cisla = {10, 20, 30, 40, 50};

int delka = cisla.Length;  // = 5
int posledni = cisla[cisla.Length - 1];  // = 50

// C# kontroluje meze pole - vyhodí výjimku při špatném indexu
int spatne = cisla[10];  // IndexOutOfRangeException!

Procházení pole

int[] cisla = {10, 20, 30, 40, 50};

// Klasický for
for (int i = 0; i < cisla.Length; i++) {
    Console.WriteLine(cisla[i]);
}

// Foreach - čitelnější, preferovaný způsob
foreach (int cislo in cisla) {
    Console.WriteLine(cislo);
}

Vícerozměrná pole

Pro data v tabulkové nebo maticové struktuře (řádky a sloupce).

2D pole v C++

// 3 řádky, 3 sloupce
int matice[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Přístup k prvkům: [řádek][sloupec]
int hodnota = matice[1][2];  // = 6 (2. řádek, 3. sloupec)

// Procházení 2D pole
for (int i = 0; i < 3; i++) {         // řádky
    for (int j = 0; j < 3; j++) {     // sloupce
        cout << matice[i][j] << " ";
    }
    cout << endl;
}
// Výstup:
// 1 2 3
// 4 5 6
// 7 8 9

2D pole v C#

// Obdélníkové pole (všechny řádky stejně dlouhé)
int[,] matice = new int[3, 3];
int[,] matice = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

int hodnota = matice[1, 2];  // = 6 (pozor: čárka, ne dva indexy)
int radky = matice.GetLength(0);    // = 3
int sloupce = matice.GetLength(1);  // = 3

// Jagged array (pole polí) - různě dlouhé řádky
int[][] jagged = new int[3][];
jagged[0] = new int[] {1, 2};
jagged[1] = new int[] {3, 4, 5, 6};
jagged[2] = new int[] {7};

int hodnota = jagged[1][2];  // = 5 (dva indexy)

Běžné operace s poli

// C# - vestavěné metody
int[] cisla = {5, 2, 8, 1, 9};

// Řazení
Array.Sort(cisla);           // = {1, 2, 5, 8, 9}
Array.Reverse(cisla);        // = {9, 8, 5, 2, 1}

// Hledání
int index = Array.IndexOf(cisla, 5);  // = 2 (nebo -1 pokud není)
bool existuje = Array.Exists(cisla, x => x > 7);  // true

// Kopírování
int[] kopie = new int[5];
Array.Copy(cisla, kopie, 5);

// Změna velikosti (vytvoří nové pole!)
Array.Resize(ref cisla, 10);  // nové prvky = 0

🎓 Shrnutí pro maturitu

  • Pole = kolekce prvků stejného typu s indexovaným přístupem
  • Indexování od 0! První prvek je pole[0]
  • C++: int pole[5]; – nekontroluje meze!
  • C#: int[] pole = new int[5]; – kontroluje meze
  • Délka: C++ = sizeof trik, C# = pole.Length
  • 2D pole: C++ = [i][j], C# obdélníkové = [i,j]
  • Pozor na index out of bounds!

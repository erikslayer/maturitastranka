📖 Co jsou datové kontejnery?

Datové kontejnery (kolekce) jsou dynamické datové struktury, které na rozdíl od polí mohou měnit svou velikost za běhu programu. Každý typ kontejneru je optimalizován pro jiné operace.

💡 Kdy použít kontejner místo pole?

  • Nevíš předem, kolik prvků budeš mít
  • Potřebuješ často přidávat/odebírat prvky
  • Potřebuješ rychlé vyhledávání podle klíče

Kontejnery v C++ (STL)

Standard Template Library (STL) poskytuje šablonové kontejnery. Hlavičkové soubory: <vector>, <list>, <map>, atd.

vector – Dynamické pole

Nejpoužívanější kontejner. Jako pole, ale automaticky zvětšuje kapacitu.

#include <vector>
using namespace std;

vector<int> cisla;              // Prázdný vector
vector<int> cisla = {1, 2, 3};  // S inicializací
vector<int> cisla(5, 0);        // 5 prvků, všechny = 0

// Přidávání prvků
cisla.push_back(10);    // Přidá na konec
cisla.push_back(20);    // {1, 2, 3, 10, 20}

// Přístup k prvkům
cout << cisla[0];       // 1 - jako u pole
cout << cisla.at(1);    // 2 - s kontrolou mezí (bezpečnější)
cout << cisla.front();  // První prvek
cout << cisla.back();   // Poslední prvek

// Velikost
cout << cisla.size();   // Počet prvků

// Procházení
for (int c : cisla) {
    cout << c << " ";
}

// Odebírání
cisla.pop_back();       // Odebere poslední prvek
cisla.clear();          // Smaže všechny prvky

list – Dvojitě spojovaný seznam

Rychlé vkládání/mazání kdekoliv, ale pomalý náhodný přístup (musí projít od začátku).

#include <list>

list<string> jmena;
jmena.push_back("Jan");     // Na konec
jmena.push_front("Eva");    // Na začátek
jmena.pop_back();           // Odebere z konce
jmena.pop_front();          // Odebere ze začátku

// Vložení doprostřed
auto it = jmena.begin();
advance(it, 1);             // Posun iterátoru
jmena.insert(it, "Petr");   // Vloží před pozici iterátoru

map – Slovník (klíč → hodnota)

Asociativní kontejner. Rychlé vyhledávání podle klíče. Klíče jsou unikátní a seřazené.

#include <map>

map<string, int> vek;       // klíč: string, hodnota: int

// Vkládání
vek["Jan"] = 25;
vek["Eva"] = 30;
vek.insert({"Petr", 28});

// Přístup
cout << vek["Jan"];         // 25

// Kontrola existence
if (vek.count("Jan") > 0) {
    cout << "Jan existuje";
}

// Procházení
for (auto& [klic, hodnota] : vek) {  // C++17
    cout << klic << ": " << hodnota << endl;
}

stack a queue – Zásobník a fronta

#include <stack>
#include <queue>

// STACK (zásobník) - LIFO: Last In, First Out
stack<int> zasobnik;
zasobnik.push(1);           // Vloží na vrchol
zasobnik.push(2);
cout << zasobnik.top();     // 2 - vrchol (neodebírá)
zasobnik.pop();             // Odebere z vrcholu
cout << zasobnik.top();     // 1

// QUEUE (fronta) - FIFO: First In, First Out
queue<string> fronta;
fronta.push("První");       // Na konec fronty
fronta.push("Druhý");
cout << fronta.front();     // "První" - začátek
fronta.pop();               // Odebere ze začátku
cout << fronta.front();     // "Druhý"

Kontejnery v C#

Namespace: System.Collections.Generic

List<T> – Dynamický seznam

using System.Collections.Generic;

List<int> cisla = new List<int>();
List<int> cisla = new List<int> { 1, 2, 3 };

// Přidávání
cisla.Add(10);              // Na konec
cisla.Insert(0, 99);        // Na pozici 0

// Přístup
int prvni = cisla[0];       // Indexování
int posledni = cisla[^1];   // C# 8: od konce

// Vyhledávání
bool obsahuje = cisla.Contains(10);      // true
int index = cisla.IndexOf(10);           // Pozice prvku

// Odebírání
cisla.Remove(10);           // Odebere první výskyt hodnoty 10
cisla.RemoveAt(0);          // Odebere na pozici 0
cisla.Clear();              // Smaže vše

// Řazení a transformace
cisla.Sort();               // Seřadí
cisla.Reverse();            // Otočí pořadí

Dictionary<TKey, TValue> – Slovník

Dictionary<string, int> vek = new Dictionary<string, int>();

// Přidání
vek["Jan"] = 25;
vek.Add("Eva", 30);         // Vyhodí výjimku pokud klíč existuje

// Přístup
int janVek = vek["Jan"];    // 25
// POZOR: pokud klíč neexistuje → KeyNotFoundException

// Bezpečný přístup
if (vek.TryGetValue("Petr", out int petrVek)) {
    Console.WriteLine(petrVek);
} else {
    Console.WriteLine("Petr neexistuje");
}

// Kontrola existence
bool existuje = vek.ContainsKey("Jan");      // true

// Procházení
foreach (var kv in vek) {
    Console.WriteLine($"{kv.Key}: {kv.Value}");
}

Queue<T> a Stack<T>

// QUEUE - Fronta (FIFO)
Queue<string> fronta = new Queue<string>();
fronta.Enqueue("První");    // Přidá na konec
fronta.Enqueue("Druhý");
string prvni = fronta.Dequeue();  // Odebere a vrátí první
string nahledy = fronta.Peek();   // Pouze nahlédne (neodebírá)

// STACK - Zásobník (LIFO)
Stack<int> zasobnik = new Stack<int>();
zasobnik.Push(1);           // Vloží na vrchol
zasobnik.Push(2);
int vrchol = zasobnik.Pop();      // Odebere a vrátí vrchol (2)
int nahledy = zasobnik.Peek();    // Nahlédne na vrchol

HashSet<T> – Množina unikátních hodnot

HashSet<string> mesta = new HashSet<string>();

mesta.Add("Praha");         // true - přidáno
mesta.Add("Brno");          // true
mesta.Add("Praha");         // false - už existuje, ignoruje se

bool obsahuje = mesta.Contains("Praha");  // Velmi rychlé!

// Množinové operace
HashSet<int> a = new HashSet<int> { 1, 2, 3 };
HashSet<int> b = new HashSet<int> { 2, 3, 4 };

a.IntersectWith(b);         // Průnik: {2, 3}
a.UnionWith(b);             // Sjednocení: {1, 2, 3, 4}
a.ExceptWith(b);            // Rozdíl: {1}

Porovnání výkonu operací

Kontejner Přístup [i] Přidání na konec Vložení doprostřed Vyhledání
vector/List O(1) ✓ O(1) O(n) O(n)
list O(n) O(1) O(1) ✓ O(n)
map/Dictionary O(log n) O(log n) - O(1) ✓
HashSet - O(1) - O(1) ✓

O(1) = konstantní čas, O(n) = lineární (roste s velikostí), O(log n) = logaritmický

🎓 Shrnutí pro maturitu

  • vector/List: Dynamické pole – rychlý přístup [i], pomalé vkládání doprostřed
  • list: Spojovaný seznam – rychlé vkládání kdekoliv, pomalý náhodný přístup
  • map/Dictionary: Klíč → hodnota, rychlé vyhledávání podle klíče
  • stack: LIFO – poslední dovnitř, první ven (push/pop)
  • queue: FIFO – první dovnitř, první ven (enqueue/dequeue)
  • HashSet: Množina unikátních hodnot, velmi rychlé Contains
